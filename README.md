# CSS Mask Builder

Инструмент для генерации CSS-масок карточек сложной формы, где размер блока зависит от реального контента, а геометрия углов остаётся предсказуемой.

## Что это

`CSS Mask Builder` решает задачу «карточки нестандартной формы без фиксированной ширины/высоты».
Вы настраиваете форму визуально, а на выходе получаете готовый CSS-код маски для встраивания в проект.

Главная идея: срезы, вырезы и скругления задаются в статичных единицах (`rem`), при этом сам элемент может расти по контенту без потери формы.

## Возможности

- Редактирование каждого угла отдельно: `TL`, `TR`, `BR`, `BL`.
- Переключаемые режимы угла:
  - `Обычный`
  - `Срез` (chamfer)
  - `Вырез` (обратный радиус с отдельной глубиной по `X` и `Y`)
  - `Радиус` (чистый радиальный вырез)
- Настройка внешнего скругления карточки.
- Настройка внутреннего скругления вырезов и скругления их внешних «плеч».
- Мини-превью углов и итоговое live-превью карточки.
- Копирование итогового CSS в один клик.

## Принцип геометрии

- Все параметры формы задаются в `rem`.
- В проекте принята шкала: `1rem = 10px`.
- Максимальное значение слайдеров: `20rem` (`200px`).
- Ширина и высота карточки не фиксируются генератором и определяются контентом/вёрсткой.

## Для чего подходит

- Карточки услуг/кейсов с «фирменными» срезами по углам.
- UI-блоки с обратными вырезами и сложным силуэтом.
- Дизайн-системы, где нужна повторяемая геометрия через CSS-переменные.

## Структура проекта

- `index.html` — интерфейс билдера.
- `styles.css` — стили интерфейса и зоны превью.
- `script.js` — логика режимов углов, параметров и генерации CSS.
